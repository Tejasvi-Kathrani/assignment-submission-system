# Assignment Submission System

A full-stack MERN application for managing and submitting assignments online.

## Features

- User authentication
- Assignment upload system
- Assignment submission tracking
- Dashboard interface
- Backend API integration
- MongoDB database connectivity

## Tech Stack

### Frontend
- React.js
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB

## Project Structure

```bash
frontend/
backend/
```

## Installation

### Clone the repository

```bash
git clone https://github.com/Tejasvi-Kathrani/assignment-submission-system.git
```

### Install dependencies

Backend:

```bash
cd backend
npm install
```

Frontend:

```bash
cd frontend
npm install
```

### Run the project

Backend:

```bash
npm start
```

Frontend:

```bash
npm start
```
